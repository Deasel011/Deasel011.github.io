Agentic engineering

In the last 3 months I have gone deep on agentic engineering. I build workflows where models and tools cooperate to turn intent into shipped code with strong guardrails.

Tooling I dabble with

  • Claude and structured prompting.
  • Codex style coding agents and repo workflows.

Models I test
